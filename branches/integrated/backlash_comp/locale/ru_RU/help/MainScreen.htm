Главный экран

Предназначение главного окна PHD2 в том, чтобы обеспечить простую и ясную последовательность действия для гидирования. Основные шаги следующие:
  1. Подключение монтировки и гидирующей камеры
  2. Запуск серийной съёмки для обнаружения пригодных для гидирования звёзд
  3. Выбор звезды для гидирования и калибровка гида
  4. Гидирование с использованием различных средств визуализации
  5. При необходимости остановка и продолжение гидирования

Большую часть окна занимает изображение звёздных полей, полученное с помощью гидирующей камеры.  Экран автоматически подстраивается под размер, яркость и контрастность, чтобы звёзды выглядели наиболее чёткими.  Однако, эти настройки относятся только к изображению.  Сам PHD2,  чтобы гидирование было наиболее эффективным, имеет дело с сырыми необработанными данными, . Экран используется также и для выбора гидируемой звезды.

Основные инструменты управления

Они находятся в нижней части экрана.  Вот из перечисление в том порядке, как они расположены - слева направо:
  1. Кнопка камеры - для подключения камеры, монтировки и дополнительного оборудования
  2. Кнопка серийной съёмки - для съёмки гидирующей камерой последовательности повторяющихся кадров, отображающихся в главном окне программы.  Нажатие на эту кнопку во время гидирования остановит его, но серийная съёмка будет продолжена.
  3. Кнопка гидирования - для калибровки (если нужно) и последующего гидирования.  
  4. Кнопка останова - для завершения гидирования и серийной съёмки
Справа от кнопки останова находится раскрывающийся список выдержек (0.01сек - 15сек).  Если камера не поддерживает выбранную выдержку, PHD2 её имитирует.  Возьмём, к примеру, веб-камеру с максимальной выдержкой 1/30 секунды.  Если выбрана секундная выдержка, PHD2 на лету автоматически сложит кадры с суммарным временем экспозиции в одну секунду.

Следующий элемент - ползунок для управления уровнем яркости.  PHD2 автоматически осуществляет грубую настройку, а ползунок может понадобиться для тонкой настройки.  Это может быть полезным, например, при фокусировке гидирующей камеры по звёздам, чтобы сделать поярче их расфокусированные изображения. Ещё раз напомним, что изменение яркости относится только к изображению, но не к данным, с которыми работает PHD2. 

Далее находится кнопка "мозг".  Она относится к  Настройкам PHD2.  Программа разрабатывалась таким образом, чтобы минимизировать необходимость их использования, однако в настройках нет ничего страшного - изучение их может заметно повысить ваш опыт, чем вам и следует заняться спустя некоторое время после начала освоения PHD2.

Крайний справа инструмент - кнопка "Камера".  В зависимости от выбранного типа гидирующей камеры, она может быть неактивной (серой).  Когда кнопка активна, с её помощью можно установить значения уникальных для камеры выбранного типа параметров - например, биннинг или настройки чувствительности.Меню

Меню

Выпадающие меню наверху главного экрана используются для доступа к различным функциям.  Подробные сведения о них содержатся в разделах 'Темновые кадры', 'Инструменты' и 'Средства визуализации'.

Панель состояния

Панель состояния внизу главного окна используется для отображения сообщений и информации для отслеживания процесса гидирования. Например, справа в трёх крайних позициях отображается информация о том, подключены ли камера и монтировка и проведена ли калибровка.
